# 02_annotation_layer

## Objective
Per-image annotation editor (arrows, boxes, text, freehand if easy) stored as SVG overlay in each image's JSON metadata. Transparent-fill support for boxes and text. Non-destructive — re-opening the editor loads editable shapes back. Overlay rendered live in editor and gallery detail view.

## Done When
User can annotate an image with arrows/boxes/text, toggle transparent vs filled on boxes/text, save, and reload the same annotations later.
