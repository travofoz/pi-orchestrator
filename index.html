what it is

Bake runs a spec-driven pipeline: an LLM executor builds code from phase specs, then ast-grep + an LLM auditor verify the result, and remediation loops until everything passes or the circuit breaker fires.

executor → ast-grep → LLM audit → remediate → ✓ / ❌

install

You need pi and ast-grep. 

pi install npm:@travofoz/pi-bake
# or project-local:
pi install -l npm:@travofoz/pi-bake

philosophy

Waterfall with the human problems removed. Sequential phases, deterministic gates (ast-grep, ~50ms), semantic verification (LLM), fast feedback loops. No meetings, no blame, no stakeholder churn. Read bake-philosophy.md for the full rationale.
